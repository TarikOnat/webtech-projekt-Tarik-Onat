## Projektbeschreibung
HTWStudyDeck ist eine Web-App zum Erstellen und Lernen von digitalen Karteikarten.  
Benutzer können Fragen und Antworten in sogenannten „Decks“ speichern  
und später im Lernmodus abfragen.

---

## Aktueller Stand (Milestone 1)
- Projektidee und Architektur definiert
- Entity-Klasse **Card** mit Feldern `id`, `question`, `answer`
- Beispiel-Controller mit GET-Route **/api/cards** implementiert
- Anwendung läuft lokal unter [http://localhost:8080/api/cards](http://localhost:8080/api/cards)
